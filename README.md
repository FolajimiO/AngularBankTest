# BankWebapp
Tech Stack Used 

- Angular v19
- Angular Material UI
- RxJS
- UUID
- SCSS / Flexbox


Tools needed to run the web app include
- Node.js - (v18  later)
- Angular CLI (npm install -g @angular/cli)



 Steps for setting up and running the application

1. Clone the repository at "https://github.com/FolajimiO/AngularBankTest.git""

   git clone https://github.com/FolajimiO/AngularBankTest.git
   

2. Install  dependencies
   -  npm install 

   - @angular/material – Material UI components (ng add @angular/material)
 
   - rxjs – Reactive programming (npm install rxjs)

   - uuid – Unique transaction IDs ( npm install uuid)


3.Run the development server
 
    ng serve

4. Open your browser and navigate to http://localhost:4200
